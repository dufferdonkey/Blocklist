# Title: no-netflix

# Description:     This host file blocks netflix.
# Formate:         Hosts

# To use this filter on windows 
# First you have to flush the DNS cache.
# Go to the command prompt and type ipconfig /flushdns
# To add in to your system
# Go to Local Disk(C)/Windows/System32/driver/etc
# You should see hosts file
# add following lines to block netflix.


0.0.0.0 netflix.com
0.0.0.0 nflxext.com
0.0.0.0 nflximg.com
0.0.0.0 netflix.es
0.0.0.0 nflxvideo.net
0.0.0.0 nflxso.net
